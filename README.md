# SIGECO - Landing Page

## 🔥 Sistema de Gestión y Coordinación de PCI

Landing page moderna para SIGECO, sistema inteligente de gestión para Protección Contra Incendios con IA predictiva y desarrollo a medida.

## 🚀 Características

- **Diseño Moderno**: Interface limpia con gradientes corporativos y animaciones suaves
- **Responsive**: Optimizado para dispositivos móviles y desktop
- **Interactivo**: Tabs, FAQ chat, formularios con validación
- **Optimizado**: Carga rápida, SEO optimizado, meta tags completos
